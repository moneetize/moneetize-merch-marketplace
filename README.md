## ✨ Features

| Feature | Description |
|---------|-------------|
| **Multi-Currency Pricing** | Toggle between USD, Tripto Tokens, and USDT |
| **Smart Cart** | Add-to-cart with quantity tracking and payment method selection |
| **Category Filtering** | 8 category cards (All Items + 7 categories) |
| **Product Tags** | Filter by New Arrivals, Best Sellers, On Sale |
| **Live Search** | Full-text search overlay |
| **Countdown Deals** | Limited-time bundle offers with live timer |
| **Testimonials** | Auto-playing carousel with manual controls |
| **Responsive Design** | Optimized for desktop, tablet, and mobile |
| **Scroll Animations** | Reveal-on-scroll with IntersectionObserver |
| **App CTA** | Download section with App Store & Google Play |

## 🚀 Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/moneetize/moneetize-merch-marketplace.git
   ```

2. Open `index.html` in your browser, or serve with any static server:
   ```bash
   npx serve .
   ```

No build tools or dependencies required — pure HTML, CSS, and JavaScript.
